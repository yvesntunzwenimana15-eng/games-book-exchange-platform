About This Project

Project Overview

SwapHub was created as an MYP Personal Project to address sustainability, accessibility, and community building among students. The platform promotes the reuse of books and games, reducing waste and fostering connections.

Learning Goals

  • Web development with HTML, CSS, JavaScript
  • Database design and management
  • User interface and experience design
  • Project planning and implementation
  • Community engagement strategies
